# homebrew-tap

Homebrew tap for [capslkagent](https://github.com/jagenaujagenau/capslkagent) —
a Claude Code agent status indicator that drives your keyboard's Caps Lock LED
(or, on Mac laptops, the keyboard backlight).

## Install

```sh
brew install jagenaujagenau/tap/capslkagent
```

Then wire it into Claude Code:

```sh
capslkagent install-hooks
```
